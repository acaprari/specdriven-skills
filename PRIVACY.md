# Privacy Policy

**specdriven-skills** is a Claude Code plugin consisting of skill files that run
locally within your Claude Code environment.

- **No data collection.** The plugin does not collect, store, or transmit any
  personal data, usage data, or telemetry.
- **No external services.** It makes no network requests and sends nothing to the
  author or any third party.
- **Local operation only.** The skills read and write spec and documentation
  files within the project you run them in. That content never leaves your
  machine through this plugin.

Any data handling that occurs is governed entirely by Claude Code and Anthropic's
own policies, not by this plugin.
